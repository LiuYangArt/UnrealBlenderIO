## 1. 项目概览

UnrealBlenderIO 是一个 Blender 插件（Blender 4.2+ / 当前仓库按 Blender 5.0 开发），用于：

1. 从 Unreal Engine 导出场景数据（JSON + FBX）到本地临时目录。
2. 在 Blender 中导入、编辑、批量处理 Actor。
3. 将编辑后的变更以 JSON 形式回传 Unreal，完成增删改同步。

仓库同时包含 UE 侧 Python 脚本与 Editor Utility Widget 资产（`UnrealAsset/`），用于在 UE 内触发导入导出。

## 7. 关键流程速查

### 7.1 Blender 插件启动
1. `__init__.py` -> `auto_load.init()`
2. `register()` 时统一注册所有类 + 注入 `Scene.ubio_params`
3. N 面板由 `UI.py` 暴露操作入口

### 7.2 UE -> Blender 导入
1. UE 侧运行 `UnrealAsset/Python/UnrealBlenderIO.py` 导出 JSON/FBX 到 `C:\Temp\UBIO\`
2. Blender 侧执行 `UBIO_OT_ImportLatestUnrealScene`/`UBIO_OT_ImportUnrealScene`
3. `UnrealBlenderIO.py::import_json_scene` 构建集合层级并恢复 Actor 信息

### 7.3 Blender -> UE 回传
1. Blender 侧执行 `UBIO_OT_ExportUnrealJSON`
2. 生成增删改 JSON（含 transform/类型/标记）
3. UE 侧 `import_json()` 读取并同步关卡 Actor

### 7.4 发布流程
1. 本地 `publish_release.bat` 调 `version_manager.py` 升级版本
2. commit + tag（`v*`）
3. GitHub Actions `release.yml` 自动打包并发布 zip

## 8. 高频改动定位建议
